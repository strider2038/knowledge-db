## Requirements
### Requirement: API удаления узла

REST API MUST предоставлять эндпоинт `DELETE /api/nodes/{path}` для удаления узла. При успешном удалении MUST возвращаться `{ "path": "<path>", "deleted": true }` со статусом 200. Если узел не найден — MUST возвращаться 404. Удаление MUST удалять главный `.md` файл узла и директорию вложений (если существует `{slug}/` рядом с файлом). Если git включён — удаление MAY быть закоммичено через GitCommitter.

#### Сценарий: Успешное удаление

- **WHEN** `DELETE /api/nodes/existing/path/my-node`
- **THEN** файл `my-node.md` и директория `my-node/` (если есть) удаляются, возвращается `{ "path": "existing/path/my-node", "deleted": true }` со статусом 200

#### Сценарий: Узел не найден

- **WHEN** `DELETE /api/nodes/nonexistent/path`
- **THEN** возвращается 404 с сообщением "node not found"

#### Сценарий: Удаление с вложениями

- **WHEN** `DELETE /api/nodes/topic/node-with-images` и существует директория `node-with-images/`
- **THEN** удаляется и файл `node-with-images.md`, и директория `node-with-images/` со всем содержимым

#### Сценарий: Пустой путь

- **WHEN** `DELETE /api/nodes` (без path)
- **THEN** возвращается 400 с сообщением "path required"

### Requirement: Модалка подтверждения удаления

UI MUST отображать модалку подтверждения перед удалением узла. Модалка MUST показывать заголовок узла и полный путь. Модалка MUST содержать кнопку «Удалить» (destructive стиль) и кнопку «Отмена». По умолчанию фокус MUST быть на кнопке «Отмена». Закрытие по Escape MUST отменять удаление.

#### Сценарий: Открытие модалки

- **WHEN** пользователь нажимает «Удалить» на панели действий
- **THEN** открывается модалка с заголовком узла, путём и кнопками «Удалить» и «Отмена»

#### Сценарий: Подтверждение удаления

- **WHEN** пользователь нажимает «Удалить» в модалке
- **THEN** вызывается `DELETE /api/nodes/{path}`, при успехе — переход на обзор, при ошибке — отображение ошибки

#### Сценарий: Отмена удаления

- **WHEN** пользователь нажимает «Отмена» или Escape
- **THEN** модалка закрывается, узел не удаляется

#### Сценарий: Индикация процесса удаления

- **WHEN** запрос на удаление выполняется
- **THEN** кнопка «Удалить» в модалке disabled, отображается спиннер
