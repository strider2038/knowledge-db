## Why

При большой иерархии тем левое дерево на странице «Обзор» показывает все уровни сразу — список становится длинным и неудобным для навигации. Нужна раскрываемая структура с предсказуемым поведением по умолчанию и сохранением состояния.

## What Changes

- Дерево тем на «Обзоре» становится **раскрываемым/сворачиваемым** по узлам (клик по теме выбирает фильтр; отдельный контрол или клик по chevron — раскрытие ветки — уточним в дизайне: обычно chevron + клик по названию выбирает).
- **Глубина по умолчанию** (сколько уровней вложенности показывать раскрытыми без ручного действия) задаётся **настройкой** в UI (число, по умолчанию 2).
- **Путь до текущей выбранной темы** (из `?path=`) **всегда раскрыт**, чтобы выбранный узел был виден.
- Состояние раскрытия и настройка глубины **сохраняются в браузере** (localStorage) и восстанавливаются после перезагрузки.
