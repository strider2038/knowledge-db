## MODIFIED Requirements

### Requirement: Панель действий на странице узла

Страница узла MUST отображать панель действий (Action Bar) — горизонтальную полосу с кнопками, расположенную после breadcrumbs и перед заголовком узла. Панель MUST содержать слева направо: toggle «Проверено» (существующий функционал), кнопку «Перевести» (для статей без перевода), кнопку «Нормализация» (для запуска нормализации текущего узла), кнопку «Обновить описание из источника» для узлов с `source_url`, разделитель, кнопку «Переместить», кнопку «Удалить». Деструктивные кнопки (Переместить, Удалить) MUST визуально отличаться (outline/ghost вместо default). Кнопка «Нормализация» MUST иметь состояние выполнения (loading/disabled) на время операции. Панель SHOULD быть sticky при прокрутке.

#### Сценарий: Отображение панели для article без перевода

- **WHEN** открыт узел типа article без переводов и с `source_url`
- **THEN** панель содержит: toggle «Проверено», кнопку «Перевести», кнопку «Нормализация», кнопку «Обновить описание из источника», разделитель, «Переместить», «Удалить»

#### Сценарий: Отображение панели для article с переводами

- **WHEN** открыт узел типа article с переводами и с `source_url`
- **THEN** панель содержит: toggle «Проверено», переключатель языков, кнопку «Нормализация», кнопку «Обновить описание из источника», разделитель, «Переместить», «Удалить» (без кнопки «Перевести»)

#### Сценарий: Отображение панели для article без source_url

- **WHEN** открыт узел типа article без `source_url`
- **THEN** панель содержит кнопку «Нормализация» и не содержит кнопку «Обновить описание из источника»

#### Сценарий: Блокировка повторного запуска нормализации

- **WHEN** пользователь нажал «Нормализация» и операция ещё выполняется
- **THEN** кнопка «Нормализация» disabled и показывает состояние выполнения до получения финального статуса

#### Сценарий: Ошибка нормализации

- **WHEN** операция нормализации завершилась ошибкой
- **THEN** UI показывает диагностируемое сообщение и возвращает кнопку «Нормализация» в обычное состояние
