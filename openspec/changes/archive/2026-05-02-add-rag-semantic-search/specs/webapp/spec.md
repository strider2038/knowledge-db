## ADDED Requirements

### Requirement: Вкладка чатбота

UI ДОЛЖЕН (SHALL) отображать вкладку «Чат» в Navbar при `KB_EMBEDDING_ENABLED=true`. Вкладка MUST содержать: поле ввода сообщения, область ответа (streaming), список источников (path + title, кликабельные). При `KB_EMBEDDING_ENABLED=false` вкладка MUST быть скрыта. UI MUST определять доступность чата через `GET /api/index/status` — при 503 вкладка скрывается.

#### Scenario: Отправка сообщения чатботу

- **WHEN** пользователь вводит сообщение и нажимает отправку
- **THEN** сообщение отправляется `POST /api/chat`, ответ отображается streaming, после источников

#### Scenario: Чатбот недоступен

- **WHEN** `GET /api/index/status` возвращает 503
- **THEN** вкладка «Чат» не отображается в Navbar

#### Scenario: Отображение источников

- **WHEN** чатбот возвращает ответ с sources
- **THEN** над текстом ответа отображаются кликабельные ссылки на ноды базы знаний (path, title)
