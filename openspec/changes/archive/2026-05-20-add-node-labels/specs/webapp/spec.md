## Purpose

Дельта: редактирование и отображение личных меток (`labels`) в веб-интерфейсе.

## ADDED Requirements

### Requirement: Ручное редактирование labels на странице узла

UI SHALL предоставлять на странице узла редактирование `labels` отдельно от `keywords`. Рядом с блоком меток MUST отображаться иконка редактирования. Нажатие MUST открывать модалку с token-input UX (аналогично keywords): chips с удалением, добавление через Enter/запятую, подсказки из GET /api/label-suggestions. Сохранение MUST отправлять PATCH с полем `labels` и обновлять страницу без перезагрузки.

#### Сценарий: Добавление метки

- **WHEN** пользователь добавляет метку «favorite» и сохраняет
- **THEN** UI отправляет PATCH с labels, отображает чип «favorite» с цветом по hash-палитре

#### Сценарий: Подсказка из существующих меток

- **WHEN** пользователь открывает модалку labels и выбирает подсказку из API
- **THEN** метка добавляется без дубликата (с учётом регистра на сервере)

#### Сценарий: Очистка всех меток

- **WHEN** пользователь удаляет все chips и сохраняет
- **THEN** UI отправляет PATCH с `labels: []`, блок меток показывает пустое состояние

#### Сценарий: Ошибка сохранения

- **WHEN** PATCH для labels возвращает ошибку валидации
- **THEN** UI показывает сообщение и оставляет модалку открытой

### Requirement: Стили чипов меток

Web UI MUST использовать общий модуль стилей меток (например `label-styles.ts`): фиксированная палитра, выбор класса по hash строки метки. Стили меток MUST NOT переиспользовать палитру типов контента (`type-styles`).

#### Сценарий: Стабильный цвет метки

- **WHEN** одна и та же метка отображается на странице узла и в таблице обзора
- **THEN** цвет чипа совпадает для этой строки метки

### Requirement: Различие keywords и labels в интерфейсе

Подписи и подсказки в UI MUST явно разделять «Ключевые слова» (семантика, поиск, RAG) и «Метки» (личная разметка, фильтр обзора).

#### Сценарий: Страница узла

- **WHEN** пользователь просматривает узел с keywords и labels
- **THEN** отображаются два отдельных блока с разными заголовками и визуальными стилями чипов
