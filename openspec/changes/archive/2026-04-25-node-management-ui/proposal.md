## Why

Страница узла (NodePage) сейчас предоставляет только просмотр и два действия: отметка «Проверено» и перевод статьи. Нет возможности удалять узлы, перемещать их между темами или управлять git-коммитами из UI. Управляющие элементы (переключатель «Проверено», кнопки перевода) разбросаны по странице без единой визуальной группы. На верхней панели нет доступа к git-операциям, хотя пользователь регулярно работает с изменениями в базе.

## What Changes

- **Панель действий на странице узла**: отдельная визуальная группа (toolbar/action bar) на странице узла, объединяющая все управляющие действия — «Проверено», «Перевести», «Удалить», «Переместить»
- **Удаление узла**: API-эндпоинт `DELETE /api/nodes/{path}` для удаления узла (файл .md и вложения); UI — модалка подтверждения перед удалением с показом заголовка и пути
- **Перемещение узла**: API-эндпоинт для перемещения узла между темами; UI — модалка с выбором целевой темы из дерева и опциональным изменением slug
- **Кнопка «Сохранить» в Navbar**: активна при незакоммиченных изменениях; неактивна при чистом дереве или при отключённом git на сервере (⚠️ + подсказка); коммит+пуш с LLM; обратная связь — toast снизу
- **API git-статуса**: эндпоинт для проверки наличия незакоммиченных изменений и эндпоинт для коммита+пуша с LLM-генерацией сообщения

## Capabilities

### New Capabilities
- `node-delete`: удаление узла из базы (API + UI с модалкой подтверждения)
- `node-move`: перемещение узла между темами (API + UI с модалкой выбора темы)
- `git-commit-ui`: кнопка «Сохранить» в Navbar, git-статус, LLM-генерация commit message, toast-уведомления
- `node-action-panel`: панель действий на странице узла (объединение управляющих элементов)

### Modified Capabilities
- `rest-api`: добавление эндпоинтов DELETE /api/nodes/{path}, POST /api/nodes/{path}/move, GET /api/git/status, POST /api/git/commit
- `webapp`: обновление NodePage (панель действий, модалки удаления/перемещения), обновление Navbar («Сохранить», toast)

## Impact

- **Backend**: `internal/api/handlers.go` — новые хендлеры; `internal/kb/store.go` — методы DeleteNode, MoveNode; `internal/ingestion/git/` — методы для git status и commit-all
- **Frontend**: `web/src/pages/NodePage.tsx` — переработка layout с панелью действий; `web/src/components/Navbar.tsx` — «Сохранить»; toast в корне; новые компоненты модалок
- **API**: 4 новых эндпоинта; изменения в роутинге (`internal/api/router.go`)
- **LLM**: использование LLM для генерации commit message (через существующую интеграцию OpenAI Responses API)
