## Purpose

### Requirement: Автоматический перевод статей

После успешного создания узла с type=article система ДОЛЖНА (SHALL) при включённой опции (KB_AUTO_TRANSLATE=true, default) проверять необходимость перевода по эвристике. Если контент преимущественно на английском — выполнять перевод на русский через LLM, создавать файл `{slug}.ru.md`, обновлять оригинал (поле `translations`, wikilink) и выполнять git commit. Перевод выполняется только для type=article; для link и note — не выполняется.

#### Сценарий: Перевод статьи при ingestion

- **WHEN** создан узел type=article с контентом на английском и KB_AUTO_TRANSLATE=true
- **THEN** система создаёт файл перевода, обновляет оригинал и выполняет git commit

#### Сценарий: Перевод отключён

- **WHEN** KB_AUTO_TRANSLATE=false
- **THEN** пост-шаг перевода не выполняется

#### Сценарий: Контент на русском

- **WHEN** эвристика определяет, что контент уже на русском
- **THEN** перевод не выполняется

#### Сценарий: Перевод только для article

- **WHEN** создан узел type=link или type=note
- **THEN** пост-шаг перевода не выполняется независимо от языка контента
