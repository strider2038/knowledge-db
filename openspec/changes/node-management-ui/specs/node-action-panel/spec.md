## Purpose

Панель действий на странице узла — единая визуальная область, объединяющая все управляющие действия: отметка проверки, перевод, перемещение и удаление.

## Requirements

### Requirement: Панель действий на странице узла

Страница узла MUST отображать панель действий (Action Bar) — горизонтальную полосу с кнопками, расположенную после breadcrumbs и перед заголовком узла. Панель MUST содержать слева направо: toggle «Проверено» (существующий функционал), кнопку «Перевести» (для статей без перевода), разделитель, кнопку «Переместить», кнопку «Удалить». Деструктивные кнопки (Переместить, Удалить) MUST визуально отличаться (outline/ghost вместо default). Панель SHOULD быть sticky при прокрутке.

#### Сценарий: Отображение панели для article без перевода

- **WHEN** открыт узел типа article без переводов
- **THEN** панель содержит: toggle «Проверено», кнопку «Перевести», разделитель, «Переместить», «Удалить»

#### Сценарий: Отображение панели для article с переводами

- **WHEN** открыт узел типа article с переводами
- **THEN** панель содержит: toggle «Проверено», переключатель языков, разделитель, «Переместить», «Удалить» (без кнопки «Перевести»)

#### Сценарий: Отображение панели для note/link

- **WHEN** открыт узел типа note или link
- **THEN** панель содержит: toggle «Проверено», разделитель, «Переместить», «Удалить» (без кнопки «Перевести»)

#### Сценарий: Отображение панели для перевода

- **WHEN** открыт перевод узла (path содержит `.`)
- **THEN** панель содержит: переключатель языков, разделитель, «Переместить», «Удалить» (без toggle «Проверено» и «Перевести»)

### Requirement: Визуальное разделение групп действий

Панель действий MUST визуально разделять кнопки на группы: основные действия (Проверено, Перевести) и деструктивные/меняющие структуру (Переместить, Удалить). Между группами MUST быть вертикальный разделитель (divider). Кнопка «Удалить» MUST использовать destructive визуальный стиль (красный цвет при hover/active).

#### Сценарий: Визуальное разделение

- **WHEN** панель действий отображается с несколькими группами
- **THEN** между группами виден вертикальный разделитель, «Удалить» имеет красный hover-стиль
